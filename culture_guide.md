## 1. EESTI MENTALITEET <a name="mentaliteet"></a>

### Põhijooned

| Joone | Kirjeldus |
|-------|-----------|
| **Vaikus** | Eestlased ei räägi lihtsalt selleks, et vaikust täita. Vaikus on normaalne. |
| **Otsekohesus** | Eestlased ütlevad otse, mida mõtlevad. See ei ole ebaviisakus — see on ausus. |
| **Täpsus** | Hilinemine on ebaviisakas. Täpsus on lugupidamine. |
| **Iseseisvus** | Eestlased hindavad iseseisvust ja privaatsust. Nad ei sega teiste asjadesse. |
| **Tagasihoidlikkus** | Enesekiitust peetakse halvaks tooniks. Parem on olla tagasihoidlik. |
| **Kannatlikkus** | Eestlased on kannatlikud ega kaeba pisiasjade üle. |
| **Kodanikuühiskond** | Eestlased usuvad reeglitesse ja seadustesse. Korruptsioon on taunitav. |
| **Sõna peab** | Lubadusi peetakse. Kui eestlane ütleb "teen", siis ta teeb. |

### Eestlased ja kommunikatsioon

| Olukord | Eestlased | Venelased (võrdluseks) |
|---------|-----------|----------------------|
| Kohtumine | Lühike "tere", nimed | Kauem, rohkem küsimusi |
| Uudised | Lühidalt, ainult faktid | Detailid, emotsioonid |
| Kaebused | Otse, ilma emotsioonita | Emotsionaalselt |
| Kiitus | Tagasihoidlik, "pole viga" | Rõõmustab, tänab pikalt |
| Hüvastijätt | Lühike, kiire | Pikk, soovid |

## 2. TERVITAMINE JA VIISAKUS <a name="tervitamine"></a>

### Kuidas tervitada

| Olukord | Kuidas? |
|---------|---------|
| Tore | "Tere!" — lühike, selge, alati sobiv |
| Hommikul | "Tere hommikust!" — kuni kella 12-ni |
| Päeval | "Tere päevast!" — kella 12–18 |
| Õhtul | "Tere õhtust!" — pärast kella 18 |
| Tuttavaga | "Tere, (nimi)!" või lihtsalt "Tere!" |
| Ametlikult | "Tere!" või "Lugupeetud..." |
| Sõbraga | "Hei!" või "Tere!" |

### Käepigistus

- Mehed suruvad kätt (lühike, tugev)
- Naisi tervitatakse samuti käepigistusega
- Lähikondseid ei tervitata põsega (erinevalt venelastest)
- Põsemuus on ainult väga lähedaste sõprade ja perega

### "Palun", "Aitäh", "Vabandust"

| Fraas | Millal kasutada |
|-------|-----------------|
| **Palun** | Iga palve ees: "Palun kohvi." "Palun istuge." |
| **Aitäh** | Iga teenuse järel: poes, kohvikus, tänaval |
| **Suur tänu** | Suurema abi eest |
| **Tänan** | Ametlikum kui "aitäh" |
| **Vabandust** | Kui kedagi segad, kui midagi valesti läheb |
| **Vabandust, palun** | Tõsise vea korral |
| **Pole tänu väärt** | Vastus "aitäh"-ile: "не стоит благодарности" |

### Praktilised fraasid

| Eesti | Vene | Millal |
|-------|------|--------|
| Palun vabandust. | Прошу прощения. | Hilinemine, viga |
| Kas tohib? | Можно? | Sisenemine, küsimus |
| Palun edasi. | Проходите, пожалуйста. | Uksel, järjekorras |
| Terviseks! | За здоровье! | Toost |
| Head isu! | Приятного аппетита! | Enne sööki |
| Head aega! | До свидания! | Lahkudes |

## 3. KINGITUSED JA KÜLASKÄIK <a name="kingitused"></a>

### Külaskäik

| Reegel | Selgitus |
|--------|----------|
| **Täpsus** | Tule täpselt kokkulepitud ajal. 5–10 minutit varem on OK. |
| **Ette teatama** | Ära mine külla ilma ette teatamata! Eestlased hindavad privaatsust. |
| **Kingitus** | Väike kingitus: lilled, vein, šokolaad, küpsetised |
| **Kingituse avamine** | Eestlased avavad kingituse tavaliselt kohe, aga vahel ka hiljem |
| **Lilled** | Paaritu arv (1, 3, 5...). Ära too 1 või 2 — toodi varem matustele. Aga tänapäeval sobib igasugune arv. |
| **Jalanõud** | Võta kingad jalast! Eestis võetakse siseruumides alati jalanõud ära. |
| **Kiitus** | Kiida kodu või sööki — see on viisakas. |

### Mida tuua ja mida mitte

| Sobib | Ei sobi |
|-------|---------|
| Lilled (1, 3, 5... või segane kimp) | Lillepott (võib tähendada probleemi) |
| Vein või hea jook | Odav vein |
| Šokolaad või komm | Kingitus, mis on liiga kallis |
| Isetehtud küpsetis | Väga isiklik kingitus (kui pole lähedane) |
| Raamat (eesti keeles) | Mitte midagi (tühi käsi) |

## 4. TÖÖKULTUUR <a name="tyokultuur"></a>

### Eesti töökultuuri põhijooned

| Reegel | Selgitus |
|--------|----------|
| **Täpsus** | Koosolek algab õigel ajal. Ära hiline. |
| **Hierarhia** | Vähe hierarhiat. Suhtutakse võrdselt, ka juhtidega. |
| **Otsused** | Otsused tehakse kiiresti ja iseseisvalt. |
| **Tagasiside** | Otsekohene, ilma emotsioonideta. Ära võta seda isiklikult. |
| **Töö ja eraelu** | Selge piir. Pärast tööaega ei olda kohustatud vastama. |
| **Puhkus** | Puhkust austatakse — tööasju ei arutata puhkuse ajal. |
| **Koosolekud** | Lühikesed ja efektiivsed. Peetakse protokolli. |
| **Riietus** | Äris ametlik (ülikond), IT-s vabam (teksad). |

### Firmakultuur

| Eestis | Erinevus Venemaaga |
|--------|-------------------|
| "Sina" vorm kasutusel kiiresti | Kauem kasutatakse "teie" |
| Otsene tagasiside | Pehmem, kaudsem |
| Vähe väikest juttu | Palju väikest juttu enne asja |
| Koosolekud lühikesed (30 min) | Pikemad (1 h+) |
| Töö-puhkus tasakaal tähtis | Töö võib olla prioriteet |

## 6. EESTLASED JA AEG <a name="aeg"></a>

### Ajataju

| Põhimõte | Tähendus |
|----------|----------|
| **Täpsus** | 5 minutit hilinemine on hilinemine |
| **Õigeaegsus** | "Koosolek algab kell 10" = algab kell 10, mitte 10:15 |
| **Planeerimine** | Eestlased plaanivad ette. Viimase hetke kutse on haruldane. |
| **Järjekord** | Järjekordi austatakse. Ära trügi ette. |
| **Aja kokkulepe** | "Helistame" = helistatakse. "Kohtume" = kohtutakse. |

### Ajaplaneerimine

| Olukord | Soovitus |
|---------|----------|
| Koosolek | Täpselt kell, kestab täpselt planeeritud aja |

| Pidu | Külalised tulevad 15–30 min pärast algust |
| Tööintervjuu | Vähemalt 5 minutit varem |
| Arst | Täpselt; kui hilined, võid aja kaotada |

## 7. EESTLASED JA LOODUS <a name="loodus"></a>

### Loodus eestlase elus

**Fakt:** Eestis on metsa 51% territooriumist. See on üks metsasemaid riike Euroopas.

| Aspekt | Kirjeldus |
|--------|-----------|
| **Mets** | Eestlased käivad palju metsas — seenel, marjul, jalutamas |
| **Suvekodu** | Paljudel on suvila või maakodu, kus veedetakse nädalavahetused |
| **Saun** | Saun on oluline osa eesti kultuurist. Peaaegu igal majal on saun. |
| **Meres ujumine** | Isegi suvel on meri jahe, aga eestlased ujuvad aasta ringi (ka talvel!) |
| **Õues elamine** | Eestlased veedavad palju aega õues — sõltumata ilmast |

### Eesti looduse reeglid

1. **Igaüheõigus** — igaüks võib minna metsa, korjata marju ja seeni, ka eramaal
2. **Lõkketegemine** — ainult selleks ettenähtud kohtades
3. **Prügi** — ära jäta prügi loodusesse (raha trahvi eest)
4. **Metsloomad** — ära toida metsloomi (karud, põdrad jne)
5. **Lilled** — ära korjaka kaitsealused lilli

## 10. HUUMOR JA SUHTLUS <a name="huumor"></a>

### Eesti huumor

| Joon | Kirjeldus |
|------|-----------|
| **Kuiv huumor** | Eestlased naljatavad tõsise näoga |
| **Iroonia** | Levinud, aga mitte solvav |
| **Eneseiroonia** | Eestlased naeravad enda üle |
| **Vaikimine** | Vahel on vaikimine ise naljakas |
| **Sõnamäng** | Levinud, eriti haritlaste seas |

### Naljakad faktid eestlaste kohta

- Eestlased ei naerata võõrastele (aga see ei tähenda, et nad on kurjad)
- Eestlased võivad seista bussipeatuses täiesti vaikselt
- "Kuidas läheb?" ei ole küsimus — piisab vastusest "Hästi"
- Eestlased teevad tööd vaikides — see on normaalne
- Eestlane ütleb mida mõtleb — ära otsi varjatud tähendust

### Eesti naljad (näited)

**Nali 1:**
Eestlane läheb arsti juurde. Arst küsib: "Mis viga on?" Eestlane: "Ei tea. Sellepärast tulingi."

**Nali 2:**
Kaks eestlast istuvad baaris. Esimene ütleb: "Tore ilm." Teine ütleb: "Jah." 30 minutit vaikust. Esimene jälle: "Eks ma lähen." Teine: "Hea küll."
