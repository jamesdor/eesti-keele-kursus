## A1 CHEAT SHEET — Kõige tähtsam

### 1. "Olema" (быть)

| Ainsus | Mitmus |
|--------|--------|
| ma **olen** | me **oleme** |
| sa **oled** | te **olete** |
| ta **on** | nad **on** |

✏️ *Ma olen õpetaja. Ta on kodus.*

### 2. Sõnajärg — VERB ON ALATI 2. KOHAL

| Vale ❌ | Õige ✅ |
|---------|---------|
| Täna **ma lähen** poodi | Täna **lähen ma** poodi |
| Eile **me käisime** kinos | Eile **käisime me** kinos |

### 3. Eitus — EI + VERB (ilma lõputa)

| Jaatus | Eitus |
|--------|-------|
| ma **tulen** | ma **ei tule** |
| sa **tead** | sa **ei tea** |
| ta **räägib** | ta **ei räägi** |

### 4. Küsimused — ALATI "Kas" alguses

✅ Kas sa tuled homme?
✅ Kas ta on kodus?
✅ Kas me läheme poodi?

### 5. Ainsus ↔ Mitmus

| Ainsus | Mitmus |
|--------|--------|
| See on **ilus maja** | Need on **ilusad majad** |
| See on **suur koer** | Need on **suured koerad** |

## A2 CHEAT SHEET

### 1. Lihtminevik (прошедшее время)

| Verb | Minevik |
|------|---------|
| tegema | ma **tegin** |
| sööma | ma **sõin** |
| jooma | ma **jõin** |
| minema | ma **läksin** |
| tulema | ma **tulin** |
| ostma | ma **ostsin** |

### 2. Osastav kääne (partitive)

| Reegel | Näide |
|--------|-------|
| kaks + **-t** | kaks **raamatut** |
| kolm + **-t** | kolm **õuna** |
| viis + **-t** | viis **eurot** |
| Ma loen + **-t** | Ma loen **raamatut** |

### 3. Kuhu? ↔ Kus? (куда? ↔ где?)

| Kuhu? (-sse) | Kus? (-s) |
|-------------|-----------|
| lähen **Tartusse** | elan **Tartus** |
| lähen **poodi** | olen **poes** |
| sõidan **linna** | olen **linnas** |

### 4. Võrdlusastmed (сравнение)

| Põhivorm | Keskvõrre | Ülivõrre |
|----------|-----------|----------|
| suur | suurem | kõige suurem |
| väike | väiksem | kõige väiksem |
| ilus | ilusam | kõige ilusam |
| hea | **parem** | kõige parem |
| halb | **halvem** | kõige halvem |

## B1 CHEAT SHEET

### 1. Kolm minevikuvormi

| Vorm | Näide | Vene |
|------|-------|------|
| **Lihtminevik** | Ma **tegin** tööd. | Я сделал работу. |
| **Täisminevik** | Ma **olen teinud** tööd. | Я сделал работу (уже). |
| **Enneminevik** | Ma **olin teinud** töö enne, kui ta tuli. | Я сделал работу до того, как он пришёл. |

### 2. Umbisikuline tegumood

| Aeg | Vorm |
|-----|------|
| Olevik | **Tehakse**, räägitakse, elatakse |
| Minevik | **Tehti**, räägiti, elati |

✅ Siin **räägitakse** eesti keelt.
✅ Eile **tehti** palju tööd.

### 3. ma- vs da-infinitiiv

| Verb + | Infinitiiv | Näide |
|--------|-----------|-------|
| tahan + | **da**-inf. | Ma tahan **magada** |
| pean + | **da**-inf. | Ma pean **minna** |
| lähen + | **ma**-inf. | Ma lähen **magama** |
| hakkan + | **ma**-inf. | Ma hakkan **tegema** |

### 4. Modaalverbid

| Verb | Tähendus | Näide |
|------|----------|-------|
| **võima** | мочь | Ma **võin** tulla |
| **saama** | мочь/получать | Ma **saan** rääkida |
| **pidama** | долженствовать | Ma **pean** minema |
| **tahtma** | хотеть | Ma **tahan** süüa |
| **tohtima** | разрешаться | Siin **tohib** suitsetada |

### 5. Siduvad sõnad

| Sõna | Vene | Näide |
|------|------|-------|
| **kes** | который (кто) | Mees, **kes** elab seal |
| **mis** | который (что) | Raamat, **mis** on laual |
| **kus** | где | Linn, **kus** ma elan |
| **mille** | которого | Maja, **mille** ostsin |

### 6. Tingiv kõneviis (сослагательное)

| Vorm | Näide |
|------|-------|
| **-ks**-lõpp | Ma **teeksin**, ta **tuleks** |
| Kui + tingiv | Kui ma **oleksin** rikas, **ostaksin** maja |

## B2 CHEAT SHEET

### 1. Umbisikuline minevik

**Reegel:** -ti, -di, -i lõpud

| Verb | Umbisik minevik |
|------|-----------------|
| tegema | **tehti** |
| rääkima | **räägiti** |
| ostma | **osteti** |
| andma | **anti** |
| tooma | **toodi** |

### 2. Kaudne kõneviis (quatative)

| Aeg | Vorm |
|-----|------|
| Olevik | ta tulevat, ta elavat, ta teadvat |
| Minevik | ta olevat tulnud, ta olevat elanud |

✅ Ta ütles, et ta **tulevat** homme.
✅ Räägitakse, et ta **olevat** Eestis käinud.

### 3. Fraasverbide põhireeglid

| Reegel | Näide |
|--------|-------|
| Verb + välja | **töötama välja, mõtlema välja, nägema välja** |
| Verb + läbi | **viima läbi, rääkima läbi, saama läbi** |
| Verb + ette | **valmistama ette, kujutama ette, lugema ette** |
| Verb + üles | **tõusma üles, leidma üles, panema üles** |
| Verb + maha | **võtma maha, jääma maha, müüma maha** |

### 4. Tingimuslaused

| Tüüp | Kui-lause | Pealause |
|------|-----------|----------|
| Reaalne | Kui **sajab**, | jään koju |
| Võimalik | Kui **sajaks**, | jääksin koju |
| Ebareaalne | Kui **oleks sadanud**, | oleksin koju jäänud |

### 5. Akadeemilised fraasid

- **Tuleb märkida, et...** — Следует отметить, что...
- **Sellest tulenevalt...** — Вследствие этого...
- **Uuringu kohaselt...** — Согласно исследованию...
- **Kokkuvõttes võib öelda...** — В заключение можно сказать...
- **Oluline on rõhutada...** — Важно подчеркнуть...

## C1 CHEAT SHEET

### 1. Lauselühendid (lühendatud kõrvallaused)

| Konstruktsioon | Näide |
|----------------|-------|
| **-des** (те, делая) | **Koju tulles** tegin süüa. |
| **-nud** (который сделал) | **Eile saabunud** kiri on laual. |
| **-tud** (сделанный) | **Kirjutatud** tekst on pikk. |
| **-v** (делающий) | **Jooksev** mees on mu vend. |
| **-tav** (делаемый) | **Loetav** raamat on huvitav. |

### 2. Kesksõna laiendamine

| Lihtne | Laiendatud |
|--------|------------|
| **jooksev mees** | **pargis jooksev mees** |
| **ostetud leib** | **eile hommikul poest ostetud leib** |
| **tulnud tudeng** | **Tartust eile tulnud tudeng** |

### 3. Sünonüümid (neutraalne → formaalne)

| Neutraalne | Formaalne |
|------------|-----------|
| palju | arvukalt, märkimisväärselt |
| oluline | määrava tähtsusega |
| näitama | demonstreerima |
| mõjutama | mõju avaldama |
| aitama | toetama |
| pärast | seejärel |
| seepärast | seetõttu |

### 4. Sõnade tuletusliited

| Liide | Tähendus | Näide |
|-------|----------|-------|
| **-ja** | tegija | õppija, töötaja, kirjutaja |
| **-mine** | tegevus | õppimine, töötamine, kirjutamine |
| **-us** | omadus | ilus → ilu**dus**, tark → tar**kus** |
| **-lik** | -ичный | õnnelik, tänulik, arusaadav |
| **-tu** | без- | kodutu, mõttetu, abitu |

### 5. Ametliku kirja struktuur

| Osa | Sisu |
|-----|------|
| **Teema** | Lühike, selge |
| **Tervitus** | Lugupeetud härra/proua/partner! |
| **Sissejuhatus** | Viide eelnevale: "Tänan teie kirja eest..." |
| **Põhiosa** | Selgitus, soov, küsimus |
| **Lõpetus** | Lugupidamisega / Parimate soovidega |
| **Allkiri** | Eesnimi + perekonnanimi, ametikoht |

### 6. Formaalne keel (ametlik)

| Mitteformaalne | Formaalne |
|---------------|-----------|
| tahan | soovin |
| palun | paluks |
| aitäh | tänan |
| aga | kuid |
| sellepärast | seetõttu |
| muidugi | kahtlemata |
| võib-olla | tõenäoliselt |
